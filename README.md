# Quiz Engine
Is a **Quiz** framework made in [GameMaker](https://gamemaker.io). With this framework you can easily made your own quiz/trivia.

Currently avaible only on Windows and Android (i don't have any other devices, so i literally can't compile builds for other platforms)

## Features
- Fully implemented quiz/trivia logic
- ~~Layout and Questions editors~~ (sorry, not implemented yet... but it will be there in future!)
- ~~Ability to make different layouts for different platforms~~ (also not implemented yet, sorry...)
- Advanced text drawing and formatting with [ScribbleDX](https://www.jujuadams.com/Scribble/#/latest/)
- Easy creation of GUI with [gooey](https://www.biyectivo.com/gooey_documentation/)

## How to compile
1. Download GameMake from [official site](https://gamemaker.io/download)
2. Open "Quiz-Engine.yyp" in GameMaker
3. Select platform (right top corner) and download required SDKs if needed
4. Click **Build** -> **Create Executable**, or click **box with arrow down** button right bellow Build and Windows buttons, or just press **Ctrl+F8**
5. If required, select build type (zip/installer)
